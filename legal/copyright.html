Re-using Wikia Content and Software

Wikia Search is dedicated to furthering the free software and copyleft movements. Only through the transparency demanded and the community input furthered by these tenets can a search project ever reach the pinnacle of quality demanded by the ever-expanding Internet user base.

With these requirements in mind, all of Wikia Searchs software, as well as the Wikia Search index, are available for your re-use.

The software which powers is licensed under the BSD license, which allows for re-use and re-distribution of the source code so long as due attribution is given to Wikia Search and the secondary use is also licensed under the BSD license. (The full terms of the license are in the LICENSE.txt file in each software package below.)

The search index, including user-initiated edits and the crawled data in raw form, are available under the Creative Commons Attribution Share-Alike 3.0 Unported license (CC-BY-SA 3.0 Unported). This license allows you to re-use and re-distribute the data so long as due attribution is given to Wikia Search and, if you make derivative works from this data, you in turn license your work under the same or compatible license. Wikia Search asks that you give attribution to us, for on-line works, by providing a hyperlink to citation to both http://search.wikia.com and to http://www.wikia.com.

You can download the materials used to make Wikia Search as follows:

Grub

Grub is a distributed web service which Wikia Search uses to crawl the web. If you wish to contribute your efforts to the Grub project, the development page is at http://dev.grub.org/ You can help by downloading the Grub client from the source repository (subversion) at http://svn.swlabs.org/grubng/. The software is available under the BSD license. If you are a developer and would like to re-use the crawled data, you can get it at http://soap.grub.org/arcs/. The crawled data is available under CC-BY-SA 3.0 Unported.

Search Tools

The software for the search user interface is available at the SVN at http://svn.swlabs.org/re.search/. If you wish to contribute your efforts to the project, please join the Search UI mailing list http://lists.wikia.com/mailman/listinfo/searchwiki.

Nutch

Nutch is an open source search engine platform which powers Wikia Search. It is a project maintained by the Apache Software Foundation. Wikia Search is working on refactoring Nutch to provide greater flexibility so that the open source community can try out different algorithms with ease. If you want to help on this project, please visit http://lucene.apache.org/nutch/mailing_lists.html and join the developers mailing list.

Index

The index used to generate the search results on Wikia Search are also available for download and re-use with due attribution, under the terms of the CC-BY-SA 3.0 Unported. You can download the index at http://search.isc.org/download/.
